# School Asset Management System (SAMS)

A modern, high-performance **Single Page Application (SPA)** designed to streamline the tracking and management of school equipment, furniture, and IT resources. 

Built with a premium "Glassmorphism" aesthetic, this system provides specialized interfaces for both Administrators and Staff members.

## ✨ Key Features

*   **Unified SPA Architecture**: Instant page transitions with zero reloads.
*   **Dual-Role Access Control**:
    *   **Admin**: Full CRUD operations, system reports, maintenance tracking, and user management.
    *   **Staff**: Browse assets, request items, and report damages.
*   **Premium UI/UX**: Glassmorphism effects, dynamic CSS variables, and high-end typography (Syne & DM Sans).
*   **Role-Based Theming**: Automatic color-scheme switching (Indigo for Admin, Emerald for Staff).
*   **Data Persistence**: Uses `localStorage` for client-side data management (No database server required for demo).
*   **Responsive Design**: Optimized for desktops and tablets.

## 🛠️ Technology Stack

*   **HTML5**: Semantic structure.
*   **CSS3**: Advanced variables, animations, and glass effects.
*   **JavaScript (ES6+)**: Core business logic and state management.
*   **Google Fonts**: Syne (Headers) & DM Sans (Body).
